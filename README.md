## Установка

- `npm install` — установка стабильной версии,
- `npm run dev` — запуск версии для разработчика,
- `npm run build` — сборка статических файлов,
- `npm run start` — сборка и запуск веб-сервера,
- `npm run test` — запуск тестов.
